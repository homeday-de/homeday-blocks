# Elevation

## Usage:

The available elevation levels are from 0 to 10.

Elevation can be used either with classes:

```js
<div class="elevation-8">My elevated block</div>
```

Or with the Sass mixin:

```css
@import '~homeday-blocks/src/styles/mixins.scss';

.my-elevated-block {
  @include elevation(8);
}
```
