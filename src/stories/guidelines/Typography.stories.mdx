# Typography

## Mixin

This mixin returns a set of typography rules for most of the typography HTML elements.

```css
@import 'homeday-blocks/src/styles/main.scss';

html {
  @include hd-typography;
  font-size: 16px;
}
```

## Text Variants mixins

The _font_ mixin allows the user to implement the text aesthetics from our current design system.

Each of the availailable variant values describes a font-size and a line-height.

```css
@import 'homeday-blocks/src/styles/main.scss';

p {
  @include font('DS-60');
}
```
